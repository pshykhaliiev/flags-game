# 🌍 Flags & Capitals

An interactive quiz game for kids to learn the flags and capitals of 120 countries around the world.

## Features

- 🎮 **120 countries** across Europe, Americas, Asia, Africa and Oceania
- 🏛 Two game modes: guess the **capital city** or the **country name**
- ❓ **20 questions** per round, randomly selected each time
- 🏆 **Shared leaderboard** — scores sync across all devices
- 📱 Works on phones, tablets and computers
- 🎨 Kid-friendly design with fun fonts and big flag emojis

## How to play

1. Enter your name
2. Choose a mode: *Capital city* or *Country name*
3. Answer 20 questions — pick from 4 choices each time
4. See your score and the global leaderboard at the end!

2. Go to **Settings → Pages**
